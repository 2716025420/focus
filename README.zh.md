# Focus - 极简专注应用

一个基于 Next.js 构建的高颜值、极简主义专注力工具。

## 🌟 核心特性

- **专注计时器**：自定义工作时长，助您保持高效。
- **4-7-8 呼吸法**：内置呼吸引导圆环，配有水滴动效，缓解焦虑，快速进入专注状态。
- **环境音效**：
  - 内置高质量音效：**单词**、**铅笔**、**敲击**。
  - 支持拖拽上传自定义音频。
- **动态背景**：随着时间流逝和模式切换，提供沉浸式的视觉体验。
- **专注记录**：统计您的专注历史，见证每一份坚持。
- **极简美学**：采用暖白色与卡其色调，提供温润、治愈的 UI 交互。

## 🛠️ 技术栈

- **框架**: [Next.js 15](https://nextjs.org/) (App Router)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **图标**: [Lucide React](https://lucide.dev/)
- **动画**: CSS Animations & Transitions

## 🚀 快速开始

1. **安装依赖**:
   ```bash
   npm install
   ```

2. **启动开发服务器**:
   ```bash
   npm run dev
   ```

3. **访问应用**:
   打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 📁 项目结构

- `src/components`: 核心功能组件（计时器、呼吸圆、音频插件等）。
- `public/audio`: 内置环境音效文件。
- `src/app`: Next.js 页面逻辑与全局样式。
