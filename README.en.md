# Focus - Minimalist Productivity App

A highly aesthetic, minimalist focus tool built with Next.js.

## 🌟 Key Features

- **Focus Timer**: Customizable sessions to help you maintain flow.
- **4-7-8 Breathing**: Guided breathing circle with water drop effects to reduce anxiety and enhance focus.
- **Ambient Audio**:
  - High-quality built-in tracks: **Words**, **Pencil**, **Tapping**.
  - Supports drag-and-drop for custom audio files.
- **Dynamic Backgrounds**: Responsive visual themes that evolve with your focus sessions.
- **Records Panel**: Track your productivity history and witness your progress.
- **Minimalist Aesthetic**: Warm white and khaki color palette, providing a gentle, healing UI experience.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animation**: CSS Animations & Transitions

## 🚀 Quick Start

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Dev Server**:
   ```bash
   npm run dev
   ```

3. **Visit App**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `src/components`: Core functional components (Timer, Breathing Circle, Audio Widget).
- `public/audio`: Built-in ambient audio assets.
- `src/app`: Page logic and global styles.
