# Focus - ミニマリスト集中アプリ

Next.js で構築された、洗練されたデザインとミニマリズムを追求した集中力向上ツール。

## 🌟 主な機能

- **集中タイマー**: カスタマイズ可能なセッションでフロー状態を維持。
- **4-7-8 呼吸法**: 水滴のようなアニメーションを伴う呼吸ガイドで不安を軽減し、集中を促進。
- **環境音**:
  - 高品質なプリセット音源：**単語**、**鉛筆**、**タイピング**。
  - ドラッグ＆ドロップによるカスタム音源の追加に対応。
- **動的背景**: 時間やモードに合わせて変化する、没入感のあるビジュアルテーマ。
- **記録パネル**: 集中した時間を記録し、日々の積み重ねを可視化。
- **ミニマリズムの美学**: 温かみのあるホワイトとカーキの配色で、癒やしを感じる UI を提供。

## 🛠️ 技術スタック

- **フレームワーク**: [Next.js 15](https://nextjs.org/) (App Router)
- **スタイリング**: [Tailwind CSS](https://tailwindcss.com/)
- **アイコン**: [Lucide React](https://lucide.dev/)
- **アニメーション**: CSS Animations & Transitions

## 🚀 クイックスタート

1. **依存関係のインストール**:
   ```bash
   npm install
   ```

2. **開発サーバーの起動**:
   ```bash
   npm run dev
   ```

3. **アプリにアクセス**:
   ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## 📁 プロジェクト構造

- `src/components`: コア機能コンポーネント（タイマー、呼吸サークル、オーディオウィジェット）。
- `public/audio`: 内蔵環境音アセット。
- `src/app`: ページロジックとグローバルスタイル。
